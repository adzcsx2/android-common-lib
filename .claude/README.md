# Claude Configuration for Android Common Library

This directory contains project-specific rules and configuration for Claude Code AI assistant.

## Purpose

These rules ensure that:
- README.md content is preserved during documentation updates
- Documentation is created in appropriate locations
- Consistent documentation style is maintained
- Project conventions are followed

## Configuration Files

### rules/documentation.md

Defines how documentation should be created and updated, including:
- README.md preservation rule (CRITICAL)
- Documentation location rules
- Style guidelines
- Review process

## Usage

These rules are automatically loaded by Claude Code when working on this project.

## Project-Specific Notes

- Documentation should be created in `/docs/` directory
- README.md should only be appended to, never modified destructively
- Follow existing documentation language conventions (Chinese for user-facing, English for technical)
