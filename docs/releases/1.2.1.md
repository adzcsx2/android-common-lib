# Version 1.2.1 Release Notes

## Release Date

March 30, 2026

## Overview

This release brings significant enhancements to ViewExtensions, Toast system, StatusBarHelper, and overall code quality improvements across multiple modules.

## Changes

### common-ui

- **ViewExtensions**: Added `selectState` binding for ImageView/View selected/unselected resource toggling, `pressEffectBgColor` / `pressEffectAlpha` / `pressEffectDisable` press effect extensions, `isDoubleClick` detection, `continuousClick` multi-tap handler, `holdClick` / `releaseClick` one-shot lock pattern, `setClickNotNull` null-safe click binding, and numerous property extensions (`backgroundColor`, `textColor`, `imageResource`, `textResource`, etc.)
- **ToastUtil**: Removed deprecated `ToastUtils`, unified toast entry point to `ToastUtil.show()` / `showLong()` / `showCenter()` / `showCenterLong()`; improved system toast handling with ROM-specific fallback (Huawei/Meizu/Xiaomi)
- **StatusBarHelper**: Added `setTransparent()`, `setLightMode()` / `setDarkMode()` for status bar icon color, `setGradientColor()` for gradient backgrounds, `getHeight()` utility
- **PressEffectHelper**: Refactored long press detection and touch boundary handling

### common-base

- **BaseViewModel**: Minor improvements to coroutine and exception handling
- **ViewModelFactory**: Enhanced ViewModel creation with improved class resolution
- **ViewModelClassResolver**: Improved generic type resolution logic
- **ViewModelExt**: New ViewModel extension functions

### common-compose

- **BaseComposeActivity**: Improved Compose lifecycle and theme integration
- **StateExtensions**: New `collectAsStateWithLifecycle` and UIState observation extensions

### common-utils

- **Logger**: Refined log filtering configuration
- **ScreenUtils**: Updated display utility methods

### common-core

- **build.gradle.kts**: Added Room KSP dependency support

### app

- Refactored data layer to DataSource pattern (`PostLocalDataSource`, `PostRemoteDataSource`)
- Added Koin DI module (`AppModule`)
- Updated demo screens to use `ViewExtensions.click` instead of raw `setOnClickListener`
- Updated all sample ViewModels and test cases

### Documentation

- Updated `claude.md` with ViewExtensions reuse rules and project-specific constraints
- Updated README, CHANGELOG, migration guide, and module checklist to `1.2.1`

## Migration from 1.2.0

### Breaking Changes

- `ToastUtils` has been removed. Replace all calls with `ToastUtil.show()` / `ToastUtil.showLong()` / `ToastUtil.showCenter()`.

### Recommended

- Replace raw `setOnClickListener` with `ViewExtensions.click { }` for built-in debounce support.
- Use `StatusBarHelper.setTransparent()` / `setLightMode()` / `setDarkMode()` instead of manual flag manipulation.
