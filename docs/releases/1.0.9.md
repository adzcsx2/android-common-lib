# Version 1.0.9 Release Notes

March 17, 2026

## Overview

This release keeps the compileSdk 34 baseline introduced in 1.0.8 and aligns the published version metadata and documentation with the `1.0.9` tag.

## Highlights

- Updated `gradle.properties` so `libVersion` matches the published `1.0.9` tag
- Added a dedicated changelog entry and migration guide section for `1.0.9`

### Documentation
- Updated README dependency snippets to `1.0.9`
- Refreshed release process examples to use tag `1.0.9`
- Kept `1.0.8` release notes as historical documentation for the compileSdk 34 baseline change
