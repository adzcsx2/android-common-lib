# Version 1.0.6 Release Notes

## Release Date

March 17, 2026

## Overview

This release refreshes the `common-image` module by renaming the Glide helper to `GlideUtils`, binding requests to `ImageView` lifecycle instead of a plain `Context`, and implementing the rounded image APIs that were previously only placeholders.

## Highlights

### Image Loading
- Renamed `ImageLoader` to `GlideUtils`
- Switched Glide request binding from `Context` to target `ImageView`
- Added rounded image loading with `CenterCrop` and `RoundedCorners`
- Added local resource overloads for circle and rounded image loading
- Standardized rounded corner examples around `10.dp` from `PxUtils`

### Quality and Documentation
- Added minimal instrumentation smoke tests for `common-image`
- Updated `common-image` API documentation and migration guidance
- Clarified the release publishing path for Android library variants with `sources.jar`

## Installation

### Single Module

```gradle
dependencies {
    implementation("com.github.adzcsx2.android-common-lib:common-image:1.0.6")

```

### All Modules

```gradle
dependencies {
    implementation("com.github.adzcsx2.android-common-lib:common-all:1.0.6")

```

## Migration Notes

- Replace `ImageLoader` imports with `GlideUtils`.
- Update direct calls from `ImageLoader.load(context, model, imageView)` to `GlideUtils.load(imageView, model)`.
- For rounded image APIs, pass pixel radius values, typically `10.dp` from `PxUtils`.
