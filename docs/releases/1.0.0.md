# Version 1.0.0 Release Notes

## Release Date

March 12, 2024

## Overview

This is the initial stable release of Android Common Library, providing a comprehensive set of utilities and abstractions for Android development.

## What's New

### Core Modules

#### common-core
- Foundation module with zero external dependencies
- Minimal footprint for maximum compatibility
- Version information access

#### common-utils
- **Context Extensions**
  - `showToast()` - Display toast messages
  - `showToast(resId)` - Display toast from resources
- **Coroutine Extensions**
  - `launchIO()` - Launch coroutine in IO dispatcher
  - `launchMain()` - Launch coroutine in Main dispatcher
  - `withIO()` - Switch to IO dispatcher
  - `withMain()` - Switch to Main dispatcher

#### common-log
- **Logger** - Unified logging interface
  - Debug/release mode support
  - Tag-based logging (v, d, i, w, e)
  - Exception logging
  - JSON formatting
  - Customizable default tag

#### common-network
- **NetworkConfig** - Centralized network configuration
  - Connect/read/write timeouts
  - Debug mode toggle
- **ApiResponse** - Generic API response wrapper
  - Success checking
  - Factory methods for success/error
- **OkHttpClientFactory** - Configurable HTTP client creation
  - Automatic logging in debug mode
  - Interceptor support
- **RetrofitFactory** - Service creation with caching
  - Reified type parameter support
  - Per-baseUrl instance caching

#### common-image
- **ImageLoader** - Glide-based image loading
  - URL and resource loading
  - RequestOptions support
  - Cache management
- **ImageView Extensions**
  - `loadImage()` - Load with placeholder/error
  - `loadCircleImage()` - Circular crop
  - `loadRoundedImage()` - Rounded corners

#### common-ui
- **BaseActivity** - ViewBinding base class
  - Automatic binding creation
  - Lifecycle hooks (initView, initData)
- **BaseFragment** - ViewBinding base class
  - Safe binding lifecycle
  - Lifecycle hooks
- **View Extensions**
  - Visibility (visible, invisible, gone, isVisible, setVisible)
  - Click handling (onClick, onLongClick)
  - Enabled state (enable, disable)

## Quick Start

```kotlin
// Initialize in Application
class MyApplication : Application() {
    override fun onCreate() {
        super.onCreate()
        Logger.init(debug = BuildConfig.DEBUG, tag = "MyApp")
        NetworkConfig.isDebug = BuildConfig.DEBUG
    }
}

// Use in Activity
class MainActivity : BaseActivity<ActivityMainBinding>() {
    override fun createBinding() =
        ActivityMainBinding.inflate(layoutInflater)

    override fun initView(savedInstanceState: Bundle?) {
        binding.submitButton.onClick { handleSubmit() }
    }

    private fun handleSubmit() {
        lifecycleScope.launchIO {
            val response = apiService.submit()
            withMain {
                if (response.isSuccess) {
                    showToast("Success!")
                }
            }
        }
    }
}
```
