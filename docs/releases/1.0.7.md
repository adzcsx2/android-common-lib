# Version 1.0.7 Release Notes

## Release Date

March 17, 2026

## Overview

This release lowers the project baseline from compileSdk 36 to compileSdk 35 while keeping the current module structure and publishing flow intact.

## Highlights

### Build Baseline
- Lowered compileSdk from 36 to 35 across the app and all library modules
- Lowered the sample app targetSdk from 36 to 35
- Downgraded `androidx.core` from `1.17.0` to `1.16.0` to keep the dependency stack compatible with compileSdk 35

### Documentation
- Updated README dependency snippets to `1.0.7`
- Updated API documentation tables to reflect the compileSdk 35 baseline
- Added migration notes for upgrading to 1.0.7

## Installation

```gradle
dependencies {
    implementation("com.github.adzcsx2.android-common-lib:common-all:1.0.7")
}
```
