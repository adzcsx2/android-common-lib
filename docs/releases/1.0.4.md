# Version 1.0.4 Release Notes

## Release Date

March 17, 2026

## Overview

This release fixes external consumption for the published Android library modules and adds an official all-in-one aggregate module for projects that want a single dependency.

## Highlights

### Publishing and Coordinates
- Fixed published Android library artifacts to generate real `aar` outputs
- Unified JitPack coordinates to `com.github.adzcsx2.android-common-lib:<module>:1.0.4`
- Added `common-all` as the supported all-in-one dependency module

### Quality and Documentation
- Added root `qualityCheck` and `libraryPublishDryRun` verification tasks
- Centralized Android test dependencies in the version catalog
- Added `common-base` and `common-compose` API docs
- Updated README and getting-started docs to distinguish single-module import from all-in-one import

### App and Demo
- Added lint baseline for historical app lint debt
- Fixed the notification permission declaration for the sample app
- Added missing LiveEvent translations in `values-zh` and `values-zh-rTW`

## Installation

### Single Module

```gradle
dependencies {
    implementation("com.github.adzcsx2.android-common-lib:common-base:1.0.4")
}
```

### All Modules

```gradle
dependencies {
    implementation("com.github.adzcsx2.android-common-lib:common-all:1.0.4")
}
```

## Notes

- `com.github.adzcsx2.android-common-lib:1.0.4` is not a valid root artifact.
- If you consumed an older broken publication, run a dependency refresh after upgrading.

## Validation

Validated with:

```bash
./gradlew :app:updateLintBaseline qualityCheck libraryPublishDryRun --stacktrace
```
