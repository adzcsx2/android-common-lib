# Version 1.0.8 Release Notes

## Release Date

March 17, 2026

## Overview

This release lowers the project baseline from compileSdk 35 to compileSdk 34 while keeping the current module structure and publishing flow intact.

## Highlights

### Build Baseline
- Lowered compileSdk from 35 to 34 across the app and all library modules
- Lowered the sample app targetSdk from 35 to 34
- Adjusted the AndroidX and Compose dependency stack to a compileSdk 34 compatible set

### Dependency Alignment
- Downgraded `androidx.core` from `1.16.0` to `1.13.1`
- Downgraded `androidx.activity` from `1.10.1` to `1.8.2`
- Downgraded `androidx.lifecycle` compose-related line to `2.8.7`
- Downgraded `androidx.navigation` from `2.9.7` to `2.8.9`
- Downgraded Compose BOM from `2025.02.00` to `2024.04.01`

### Documentation
- Updated README dependency snippets to `1.0.8`
- Updated API documentation tables to reflect the compileSdk 34 baseline
- Added migration notes for upgrading to 1.0.8

## Installation

```gradle
dependencies {
    implementation("com.github.adzcsx2.android-common-lib:common-all:1.0.8")
}
```
