# Version 1.2.0 Release Notes

## Release Date

March 19, 2026

## Overview

This release introduces AI-assisted development support with comprehensive development guidelines and module functionality checklist.

## Highlights

### New Files
- **`claude.md`** - Project development guidelines for AI tools (Claude Code, Cursor, GitHub Copilot)
  - Clean Architecture principles
  - Module dependency hierarchy
  - Code standards (Koin DI, Repository pattern, etc.)
  - Common mistakes reference table

- **`docs/checklist/modules.md`** - Comprehensive module functionality checklist
  - Network requests (common-network)
  - Image loading (common-image)
  - State management (common-core)
  - Logging (common-utils)
  - UI helpers (common-ui)
  - Architecture base classes (common-base)
  - Compose support (common-compose)

### Documentation Enhancements
- **README.md** new sections:
  - AI Assisted Development - How to configure AI tools
  - Architecture Details - Clean Architecture layering explanation
  - Updated module checklist references

- **.gitignore** updated:
  - Added `.claude/` directory ignore rule (keeping `claude.md` in repo)

## Installation

```gradle
dependencies {
    // All modules
    implementation("com.github.adzcsx2.android-common-lib:common-all:1.2.0")

    // Or individual modules
    implementation("com.github.adzcsx2.android-common-lib:common-core:1.2.0")
    implementation("com.github.adzcsx2.android-common-lib:common-base:1.2.0")
    implementation("com.github.adzcsx2.android-common-lib:common-utils:1.2.0")
    implementation("com.github.adzcsx2.android-common-lib:common-network:1.2.0")
    implementation("com.github.adzcsx2.android-common-lib:common-image:1.2.0")
    implementation("com.github.adzcsx2.android-common-lib:common-ui:1.2.0")
    implementation("com.github.adzcsx2.android-common-lib:common-compose:1.2.0")
}
```

## AI Development Setup

When using AI tools, copy the guidelines to your project:

```bash
# Copy development guidelines
cp /path/to/common-lib/claude.md /path/to/your-project/
cp /path/to/common-lib/docs/checklist/modules.md /path/to/your-project/
```

## Migration from 1.1.0

No breaking changes. This release only adds documentation and AI development support.
